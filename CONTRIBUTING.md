# 贡献指南

感谢你帮助改进本项目。

## 可以贡献什么

- 报告学习通页面结构变化导致的解析失败；
- 补充题型、图片、答案或难度映射兼容；
- 修复考试宝 Excel 导入问题；
- 改进性能、可访问性和安全性；
- 完善文档、测试和错误提示；
- 提供原始项目地址及许可证线索。

## 报告问题前

1. 确认使用最新版本；
2. 刷新页面后重新解析；
3. 打开浏览器开发者工具查看控制台错误；
4. 对截图、HTML 和日志中的姓名、学号、课程名、账号、Cookie、Token 和 API 密钥进行脱敏；
5. 搜索是否已有相同 Issue。

## Pull Request 流程

1. 从 `main` 创建功能分支，例如 `fix/classroom-image-export`；
2. 保持改动聚焦，不在同一个 PR 中混入无关格式化；
3. 更新相关文档和 `CHANGELOG.md`；
4. 若修改版本号，同时更新用户脚本元数据、`package.json` 与 `CITATION.cff`；
5. 本地运行：

```bash
npm run check
```

6. 在 PR 描述中说明测试页面类型、浏览器、用户脚本管理器和预期结果。

## 代码约定

- 使用 UTF-8、LF 换行和 4 空格缩进；
- 优先使用明确的函数名和结构化数据，避免继续扩大隐式全局状态；
- 所有 DOM 选择器变更需说明对应页面结构；
- 不提交账号、Cookie、Token、API 密钥、真实题库或含个人信息的页面存档；
- 新增网络请求时必须同步评估并收紧 `@connect` 权限；
- 不引入自动提交答案、绕过考试限制或规避访问控制的功能。

## 许可注意事项

原始许可证仍在核查中。提交贡献即表示你有权提交相关代码，并同意在未来原始许可证确认后，配合采用与其兼容的项目许可方式。在许可状态明确前，请不要复制来源不清的第三方代码。
