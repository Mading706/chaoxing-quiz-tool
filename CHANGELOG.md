# 更新日志

本项目采用语义化版本号形式记录维护版本。由于原始发布历史尚未完整找到，早期改动仅依据当前脚本中的版本信息整理。

## [1.46.0] - 2026-07-05

### Added

- 建立规范化 GitHub 仓库结构；
- 增加 README、贡献指南、安全说明、来源核查记录和引用元数据；
- 配置 GitHub Actions 语法与元数据检查；
- 配置基于 Git 标签的 Release 工作流；
- 增加 GitHub Raw 自动更新地址。

### Changed

- 规范用户脚本名称、命名空间、主页、支持地址和维护者署名；
- 将版本号规范为 `1.46.0`；
- 删除对任意网站中 `exam` 或 `test` 路径的过宽匹配，仅保留学习通域名；
- 将 `@connect *` 收紧为学习通及可选 AI 服务域名；
- 优先使用用户脚本管理器隔离存储保存设置和 API 密钥，并兼容迁移旧版 `localStorage` 配置。

### Existing capabilities consolidated in this version

- 普通作业、考试、随堂练习和随堂练习统计详情解析；
- 考试宝 Excel、普通 Excel、Word、兼容 Word 和 PDF 导出；
- 题干及选项图片处理；
- 答案、解析、知识点和难度提取；
- 可选 AI 单题及错题解析。
