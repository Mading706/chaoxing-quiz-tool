# 安全政策

## 支持范围

安全修复优先针对 `main` 分支最新版本。

## 报告安全问题

请不要在公开 Issue 中披露以下内容：

- API 密钥、Cookie、Token 或账号凭据；
- 可导致任意代码执行或敏感信息泄露的完整利用步骤；
- 含学生、教师或课程个人信息的未脱敏页面数据。

请通过 GitHub 仓库所有者公开资料中的联系方式进行私下报告，并提供：

- 受影响版本；
- 问题类型及影响；
- 最小复现步骤；
- 建议修复方式；
- 是否已经公开披露。

## 已知安全边界

- AI 功能会把所选题目发送给用户配置的第三方模型服务商；
- 脚本加载 SheetJS、ExcelJS、jsPDF 和 html2canvas 等第三方 CDN 依赖；
- 图片下载需要跨域请求权限；
- 导出内容可能包含课程材料和个人作答记录；
- 用户应当只在可信浏览器环境中保存 API 密钥。

从本仓库版本开始，设置优先保存在用户脚本管理器隔离存储中，并已移除任意网站匹配及全域 `@connect *` 权限。
